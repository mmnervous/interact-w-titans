# interact-w-titans

## Compile
`gcc main.c utils.c int_request/* -lcurl -lgmp -o file_name`

## Run
`./file_name`

## Option
1 = getBalance <br>
2 = unForbidden <br>
3 = unlockAccont <br>
4 = withdrawReward <br>

## Example

./file_name 1 publicAddress <br>
./file_name 2 publicAddress <br>
./file_name 3 publicAddres password <br>
./file_name 4 fromAddress delegateAddress <br>
